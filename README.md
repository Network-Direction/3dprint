# 3dprint
My repository of 3d printing information and models.

This is focused on the Ender 3 v3, but would be helpful for other printers too.


</br></br>
## Klipper

Information about using Klipper, including how to connect a Raspberry Pi to the printer, and flash it with Klipper firmware.


</br></br>
## Calibration

Calibrating and tuning Klipper settings for the best prints.


</br></br>
## Macros

Macros used in Klipper or Octoprint.


</br></br>
## Models

Useful models, including Autodesk Fusion source files.
