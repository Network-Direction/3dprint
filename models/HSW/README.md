# Honeycomb Storage Wall (HSW)

This is a collection of models and files related to the project found here: [https://makerworld.com/en/models/22521-honeycomb-storage-wall]


</br></br>
## Files

Files are available in various formats.

| File Type | Usage                                                         |
| --------- | ------------------------------------------------------------- | 
| stl       | The raw model. Will need to be added to a slicer              |
| 3mf       | A slicer project file. Uses settings suitable for an Ender 3  |
| f3d       | An Autodesk Fusion file. So you can edit the designs yourself |
