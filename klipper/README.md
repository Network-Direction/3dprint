# Klipper Guide

The files in this section contain the guide to setup your printer with Klipper, and to connect a slicer to it.

Start with [the Klipper guide](klipper.md) and then move on to [the slicer configuration](orca_setup.md).

Optionally you can use [the slicer settings](slicer_settings.md) that I have created, or use your own.

You may also want to try to [restore the LCD screen functionality](ender_screen.md) after the update.
