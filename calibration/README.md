# Calibration Guide

This is my calibration guide.

While it should work on other printers, it assumes an Ender 3 v3 SE, with the Klipper firmware.


## Process

Calibration should be performed in this order:

1. Rotation Distance
2. Heater PID Tuning
3. Z-Offset
4. Bed Leveling (AKA 'Bed Mesh')
5. Axis Twist Compensation
6. Resonance Compensation
7. Pressure Advance
8. Flow Rate
