# Calibration Guide

This is my calibration guide.

While it should work on other printers, it assumes an Ender 3 v3 SE, with the Klipper firmware.


## Process

Calibration should be performed in this order:

1. Approximate bed level
2. Rotation Distance
3. Heater PID Tuning
4. Axis Twist Compensation
5. Z-Offset
6. Bed Leveling (AKA 'Bed Mesh')
7. Resonance Compensation (AKA 'Input Shaper')
8. Pressure Advance
9. Flow Rate
